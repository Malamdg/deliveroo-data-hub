# Vision

Deliveroo Data Hub aims to help users reclaim their personal consumption data
and gain better visibility over their habits.

## Principles

- Simplicity over complexity
- User-first experience (not developer-first)
- Transparency of data collection
- No lock-in: data is exportable and reusable

## Long-term direction

- Provide meaningful insights on spending habits
- Enable better financial awareness
- Stay lightweight and non-intrusive (userscript / extension)
