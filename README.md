# Deliveroo Data Hub

Extract, analyze, and export your Deliveroo order history directly from your browser.

A client-side tool to retrieve and export your Deliveroo order history without relying on external APIs.

## ✨ Features (V1)

- Extract full Deliveroo order history
- Filter by date range
- Export to JSON
- Works directly in the browser (userscript)

## 🚀 Vision

This project aims to help users take control of their food delivery data and gain insights into their consumption habits.

## ⚠️ Disclaimer

This tool runs entirely on the client side and requires the user to be logged into Deliveroo.
